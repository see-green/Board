# 取締役会 - ゲーム資金管理



あなたはお金を数えることを心配することなくあなたがゲームを楽しむことができるように、我々のアプリはあなたがモノポリーの面白い面に集中してあなたのためにすべてのあなたのお金の取引を扱うことを可能にします。 あなたは何を待っていますか？ 是非、私たちのアプリを試してみてください。

あなたはまだゲームをプレイしながらお金の取引を記録することができないことを心配していますか？ あなたがあなたを助けることができるアプリケーションを探しているなら、あなたは私たちのアプリを使うことができます。 これは模擬ビジネスゲームでお金を管理するのに役立つアプリです。ゲームでのお金の取引について心配する必要がなくなり、ゲームに集中することができます。

機能：
*プレーヤーを追加するには、（+）ボタンをクリックします。
*最大6人のプレイヤーのアカウントを追跡します。
*アプリ内のすべての現金取引を処理します。
*プレイヤーや銀行間でドラッグアンドドロップして資金を移動します。
